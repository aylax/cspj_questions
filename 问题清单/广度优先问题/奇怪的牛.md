## 奇怪的牛

### 题目描述

农夫知道一头牛的位置，想要抓住它。农夫和牛都位于数轴上，农夫起始位于点 $N$ ，牛位于点 $K$ 。农夫有两种移动方式：

$1$：从 $X$ 移动到 $X-1$ 或 $X+1$，每次移动花费一分钟。

$2$：从 $X$ 移动到 $2 \times X$，每次移动花费一分钟。

假设牛没有意识到农夫的行动，站在原地不动。农夫最少要花多少时间才能抓住牛？

### 输入格式

两个整数，$N$ 和 $K$。

### 输出格式

一个整数，农夫抓到牛所要花费的最小分钟数。

### 样例

#### 样例输入

```
5 17
```

#### 样例输出

```
4
```


### 提示


$0 \le N \le 100000$

$0 \le K \le 100000$
