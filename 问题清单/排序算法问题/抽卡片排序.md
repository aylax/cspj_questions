## 抽卡片排序

### 题目描述

小红和小明正在玩一个游戏，游戏规则如下：<br>
现在有 $n$ 张卡片摞在桌子上，每张卡片上有一个正整数编号（每张卡片的编号不同，且属于 $1～n$），<br>现在每次可以将一张卡片抽出并放在最上面，请问最少多少次可以使得从上到下的卡片的编号呈升序。

### 输入格式

第一行为 $n$。第二行开始为 $n$ 个正整数的值，整数间用空格隔开，表示从上到下每个卡牌的编号。

### 输出格式

输出最小操作次数。

### 样例

#### 样例输入

```
5
1 4 5 3 2
```

### 样例输出

```
3
```

### 提示

${1 \le n \le 10^{5}}$
