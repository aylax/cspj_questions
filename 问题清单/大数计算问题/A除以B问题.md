## $A \div B$ $Problem$

### 题目描述

给定两个正整数 $A$ 和 $B$ ，输出 $A \div B$ 的值。{ $A/B$  的下取整 }

### 输入格式

输入共两行，每行包含一个正整数。

### 输出格式

输出 $A \div B$ 的结果值。

### 样例

#### 样例输入

```
1000
333
```

#### 样例输出

```
3
```

### 提示

对于 $100 \%$ 的数据，$A, B$ 均在 $10^{200}$ 范围内，且 $A + B \ge 0$。
