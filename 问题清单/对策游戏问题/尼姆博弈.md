## 尼姆博弈

### 题目描述

有 $n$ 堆石子，每一堆的数量任意。游戏开始由两个人 $Alice$ 和 $Bob$ 轮流取石子。

游戏规定, 每人轮流，在任意的一堆中，取走任意多的石子，至少取一个。最后把石子全部取完的人获胜。

假设 $Alice$先取, 双方都采用最优策略，请问谁最终会获胜?


### 输入格式

共两行

第一行一个整数 $n$，表示堆数

第二行共 $n$ 个整数, $n_i$ 表示第 $i$ 堆的数量。

### 输出格式

一行, 胜利方

### 样例

#### 样例输入

```
3
1 2 3
```

#### 样例输出

```
Bob
```


### 提示

$ 1 \le n $

$ 0 \le n_i $
