## 巴什博弈

### 题目描述

有一堆石子共 $n$ 颗, $Alice$ 和 $Bob$ 轮流从石子堆中取若干枚石子, 每次至少取 $1$枚, 最多取 $k$枚, 最后一次取完石子的获胜。

假设 $Alice$先取, 双方都采用最优策略，请问谁最终会获胜?


### 输入格式

一行，两个用空格分隔的整数，$N$ 和 $K$。

### 输出格式

一行, 胜利方

### 样例

#### 样例输入

```
4 2
```

#### 样例输出

```
Alice
```


### 提示


$1 \le k \lt n$
