## 第一个只出现一次的字符

### 题目描述

在一个字符串中找到第一个只出现一次的字符。如输入 `abaccdeff`，则输出 `b` 。

### 输入格式

一行字符串, 输入的字符串满足以下条件：

1. 字符都是合法ASCII码表字符

2. 字符串中不包含空格

3. 至少存在一个只出现一次的字符

### 输出格式

输出符合条件的字符

### 样例

#### 样例输入

```
believable
```

#### 样例输出

```
i
```

### 提示

```
ASCII码表共有256个字符, 其中:
0 ~ 31 & 127 功能字符,
32 ~ 126 打印字符,
128 ~ 255 扩展字符
```
