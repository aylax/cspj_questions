## 字符串匹配问题

### 题目描述

字符串中只含有括号 (),[],&lt;&gt;,{}。

判断输入的字符串中括号是否匹配。如果括号有互相包含的形式，从内到外必须是&lt;&gt;,(),[],{}。

例如。输入: [()] 输出:YES，而输入([])， ([])都应该输出NO。

### 输入格式

文件的第一行为一个整数n(0&lt;n&lt;20)，表示以下有多少个由括号组成的字符串。接下来的n行，每行都是一个由括号组成的长度不超过255的字符串。

### 输出格式

在输出文件中有N行，每行都是YES或NO。

### 样例

#### 样例输入

```
5
{}{}<><>()()[][]
{{}}{{}}<<>><<>>(())(())[[]][[]]
{{}}{{}}<<>><<>>(())(())[[]][[]]
{<>}{[]}<<<>><<>>>((<>))(())[[(<>)]][[]]
><}{{[]}<<<>><<>>>((<>))(())[[(<>)]][[]]
```

#### 样例输出

```
YES
YES
YES
YES
NO
```
