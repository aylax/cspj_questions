## 求二叉树前序2

### 题目描述

输入一串二叉树，输出其前序遍历。

### 输入格式

第一行为二叉树的节点数 $n$。

后面 $n$ 行，每一个字母为节点，后两个字母分别为其左右儿子。特别地，数据保证第一行读入的节点必为根节点。

空节点用 `*` 表示

### 输出格式

二叉树的前序遍历。

### 样例

#### 样例输入

```
6
abc
bdi
cj*
d**
i**
j**
```

#### 样例输出

```
abdicj
```

### 提示

$1 \le n \le 26$
